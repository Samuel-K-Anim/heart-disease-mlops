# heart-disease-mlops

Repository scaffold for a heart disease MLOps workflow:

- `.github/workflows/` for CI/CD
- `config/` for model and Feast configuration
- `data/` for local dumps (sample tracked)
- `feature_store/` for Feast repository configuration
- `src/` for ingestion, training, fine-tuning, evaluation, and API app code
- `tests/` for unit tests
